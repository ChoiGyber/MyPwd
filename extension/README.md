# MyPwd Browser Extension

## Installation (Development)

1. Open Chrome/Edge and go to `chrome://extensions/` (or `edge://extensions/`)
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `extension` folder from this project

## Icons
Replace the placeholder icons in `icons/` with proper PNG files:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## How it works
1. Start the MyPwd desktop app and unlock it
2. Visit a website with a login form
3. Click the lock icon next to the password field, or click the extension icon in the toolbar
4. Select the credential to auto-fill
